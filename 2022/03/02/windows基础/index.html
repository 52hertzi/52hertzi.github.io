Windows

——内网开始

基础命令

  • regedit——打开注册表
  • eventvwr——日志查看器
  • gpedit.msc——组策略编辑器
  • services.msc——服务查看器

权限划分

权限由高到低

trustedinstaller–>system–>administrators–>user

权限继承

administrator管理员运行iis–>iis获得administrator权限

默认账户

  • administrator
  • guest——默认禁用

用户组

基础知识

  • 组是多个用户的集合
  • 一个用户可以隶属于多个组
  • 用户加入组后，会获得相应权限
  • 给予一个组对应的权限后，组内所有用户都会获得该权限

重点组

  • administrators——管理员组——内置：administrator（域管理会默认加入到本地管理员组）
  • guests——来宾组——内置：Guest
  • Users——用户组
  • Remote Desktop Users——远程桌面组——远程登录权限（mstsc）
  • Print User——打印机用户组——管理打印机

SID：安全描述符

  • 唯一，标识用户、组、计算机账户——命令：whoami /user

用户命令

  • net user——查看用户
  • net user + 用户名——查看指定用户
  • query user——查看当前在线用户
  • whoami ——查看权限 格式 主机名+用户名
  • whoami /priv—— 查看详细权限
  • net user + 用户名 + password /add——添加用户
  • net user + 用户名 /del——删除用户
  • logoff + id——注销用户

用户组命令

  • net localgroup——查看本地组
  • net localgroup group_name——查看组成员
  • net localgroup +组名+用户名 /add ——将用户添加到指定组
  • net localgroup +组名+用户名 /del——将用户从指定组中移除
